# Spring  

**AOP 植入切面的方法：**

	1. 编译期织入，这要求使用特殊的Java编译器，AspectJ是其中的代表者；
	2. 类装载期织入，而这要求使用特殊的类装载器，AspectJ和AspectWerkz是其中的代表者；
	3. 动态代理织入，在运行期为目标类添加增强生成子类的方式，spring AOP采用动态代理织入切面。


**Spring AOP使用了两种代理机制：** 

	1. 基于JDK的动态代理，
	2. 基于CGLib的动态代理， 

之所以需要两种代理机制，很大程度上是因为JDK本身只提供基于接口的代理，不支持类的代理。
